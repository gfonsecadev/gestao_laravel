# Super Gestão
Esta aplicação foi desenvolvida utilizando o Framework Laravel 7.30.6 com php na versão 7.2.5. Esta aplicação foi desenvolvida com intuito de gerenciar uma loja, com opções de cadastrar fornecedores, produtos, clientes e pedidos de clientes.
